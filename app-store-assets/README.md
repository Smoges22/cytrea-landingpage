# Cytrea App Store Assets

This folder contains the final App Store screenshot exports for Cytrea.

- Screenshot exports: `app-store-assets/screenshots/`
- Export size: 1290 x 2796 PNG
- Source UI: current Cytrea production screenshot mockups from `images/mockups/`
- Copy reference: `screenshot-copy.md`
- Order reference: `screenshot-plan.md`

No image-generation model was used for these exports.
